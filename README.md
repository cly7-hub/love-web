# 💕 情侣网页 - 恋爱纪念网站

一个专为情侣打造的浪漫纪念网页，记录美好回忆，见证爱情成长。

## ✨ 功能特点

### 🏠 首页模块
- **视觉冲击Banner**：全屏背景图 + 情侣昵称 + 浪漫Slogan
- **实时时间计数器**：精确显示恋爱时长（天/时/分/秒）
- **纪念日倒计时**：显示距离下次纪念日的时间
- **动态特效**：花瓣飘落、星空闪烁、背景缩放
- **核心入口**：快速进入各个功能模块

### 💖 恋爱时间线
- **时间轴设计**：垂直滚动，记录每个重要时刻
- **节点交互**：点击展开详情，支持照片、文字、地点
- **添加节点**：可以随时添加新的纪念时刻
- **分类图标**：区分遇见、告白、旅行、生日等类型

### 📸 回忆相册
- **3D背景特效**：粒子系统 + 连线效果
- **照片墙滑动**：支持鼠标拖拽和触摸滑动
- **优化的3D爱心**：多层旋转爱心，带有光晕效果
- **分类筛选**：按日常、旅行、纪念日、美食等分类
- **全屏查看**：点击照片进入全屏模式，支持左右切换
- **照片上传**：支持批量上传照片

### ✅ 互动打卡
- **日常打卡**：今日思念、一起做的事
- **挑战打卡**：30天早安晚安、一起读书等任务
- **打卡日历**：按月展示，直观查看打卡记录
- **进度追踪**：实时显示挑战进度
- **成功提示**：打卡成功时的动画提示

### ⭐ 愿望清单
- **分类管理**：短期愿望、长期愿望、旅行清单
- **状态标记**：未完成、进行中、已实现
- **卡片设计**：每个愿望独立卡片，支持拖拽排序
- **完成勋章**：已实现的愿望显示完成标识

### 💑 情侣专属
- **专属昵称墙**：展示双方给彼此的所有昵称
- **恋爱条约**：自定义情侣规则，手写风格
- **私密留言板**：仅情侣可见的悄悄话
- **专属资源**：壁纸、表情包、语录生成器

### 📖 关于我们
- **故事章节**：分章节讲述爱情故事
- **情侣档案**：展示双方基本信息、性格特点
- **爱情宣言**：共同宣言 + 手写签名

### ⚙️ 设置功能
- **隐私设置**：公开/仅情侣可见/密码访问
- **主题切换**：温柔粉、治愈蓝、森系绿
- **数据备份**：导出照片、时间线、愿望清单

## 🎨 设计风格

- **配色方案**：温柔治愈（浅粉 #FFD1DC + 浅蓝 #B3E5FC + 米白 #F5F5F5）
- **字体选择**：圆润标题 + 清晰正文

## 🚀 使用方法

### 1. 修改个人信息

打开 `index.html`，修改以下内容：

```html
<!-- 修改情侣昵称 -->
<h1 class="couple-title">XX & XX</h1>

<!-- 修改Slogan -->
<p class="couple-slogan">从遇见你的那天起，风都变甜了</p>
```

### 2. 设置恋爱开始时间

打开 `js/main.js`，修改：

```javascript
// 修改为实际的恋爱开始时间
const loveStartDate = new Date('2023-02-14T00:00:00');

// 修改下次纪念日
const nextAnniversaryDate = new Date('2024-05-20T00:00:00');
```

### 3. 添加照片

将照片放入 `3D爱心/images/` 目录，或修改 `js/gallery.js` 中的照片路径。

### 4. 自定义内容

- **时间线**：在网页上点击"添加新节点"即可添加
- **愿望清单**：可以手动修改HTML或通过JavaScript动态添加
- **关于我们**：编辑 `index.html` 中的对应章节

- 如需在服务器上运行，需要配置HTTP服务器（不能用file://协议）
