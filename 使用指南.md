# 💕 情侣网页 - 快速使用指南

## 🚀 快速开始

### 第一步：打开网页
直接在浏览器中打开 `index.html` 文件即可。

### 第二步：修改个人信息

#### 1. 修改情侣昵称和Slogan
打开 `index.html`，找到以下内容并修改：

```html
<h1 class="couple-title">XX & XX</h1>  <!-- 改为你们的名字 -->
<p class="couple-slogan">从遇见你的那天起，风都变甜了</p>  <!-- 改为你们的专属Slogan -->
```

#### 2. 设置恋爱开始时间
打开 `js/main.js`，找到并修改：

```javascript
// 改为你们在一起的第一天
const loveStartDate = new Date('2023-02-14T00:00:00');

// 改为下一个纪念日
const nextAnniversaryDate = new Date('2024-05-20T00:00:00');
```

#### 3. 修改关于我们
在 `index.html` 中找到"关于我们"部分，修改：
- 双方的基本信息（星座、爱好、性格等）
- 爱情故事章节
- 爱情宣言

## 📸 添加照片

### 方法一：使用现有文件夹
1. 将照片放入 `3D爱心/images/` 文件夹
2. 照片会自动显示在相册中

### 方法二：通过网页上传
1. 进入"回忆相册"模块
2. 点击"+ 上传照片"按钮
3. 选择照片即可上传

### 方法三：修改照片数据
编辑 `js/gallery.js` 中的 `galleryPhotos` 数组：

```javascript
const galleryPhotos = [
    { 
        id: 1, 
        url: '你的照片路径',  // 改为照片路径
        category: 'daily',    // 分类：daily/travel/anniversary/food
        date: '2023.05.20', 
        location: 'XX餐厅', 
        note: '第一次一起吃烛光晚餐' 
    },
    // 添加更多照片...
];
```

## 💖 添加时间线节点

### 方法一：通过网页添加
1. 滚动到"恋爱时间线"模块
2. 点击"+ 添加新节点"按钮
3. 填写信息并提交

### 方法二：直接修改代码
编辑 `js/timeline.js` 中的 `timelineData` 数组

## ⭐ 添加愿望

1. 进入"愿望清单"模块
2. 点击"+ 添加愿望"按钮
3. 填写愿望信息并选择分类
4. 可以随时修改状态（未完成/进行中/已实现）

## ✅ 使用打卡功能

1. 进入"互动打卡"模块
2. 选择"日常打卡"标签
3. 填写"今日思念"或勾选"一起做的事"
4. 切换到"打卡日历"查看记录

## 🎨 更换主题

1. 点击底部导航的"设置"
2. 在"主题设置"中选择：
   - 温柔粉（默认）
   - 治愈蓝
   - 森系绿

## 💾 数据备份

1. 点击底部导航的"设置"
2. 点击"导出数据"按钮
3. 数据会以JSON格式下载到本地

⚠️ **重要提示**：所有数据都保存在浏览器本地，清除浏览器数据会丢失所有内容。建议定期备份！

## ❓ 常见问题

### Q: 照片不显示怎么办？
A: 检查照片路径是否正确，或者使用"上传照片"功能。

### Q: 数据丢失了怎么办？
A: 如果之前导出过数据，可以重新导入（需要手动修改代码）。

### Q: 可以修改颜色吗？
A: 可以！修改 `css/style.css` 中的 CSS 变量：

```css
:root {
    --primary-pink: #FFD1DC;  /* 主色调 */
    --primary-blue: #B3E5FC;  /* 辅助色 */
}
```

### Q: 如何在服务器上运行？
A: 需要配置HTTP服务器（如Apache、Nginx），不能直接用file://协议打开。
