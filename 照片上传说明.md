# 📸 照片上传功能说明

## ✨ 功能特点

1. **最多30张照片**：相册最多可存储30张照片
2. **智能合并**：自动合并文件夹照片和上传的照片
3. **优先显示**：文件夹中的照片优先显示，上传的照片补足剩余位置
4. **删除功能**：可以删除上传的照片（文件夹照片不可删除）

## 📁 照片来源

### 1. 文件夹照片
- 位置：`3D爱心/images/` 文件夹
- 配置：在 `js/gallery.js` 的 `folderPhotos` 数组中配置路径
- 特点：这些照片会优先显示，且不可在网页中删除

### 2. 上传的照片
- 存储位置：浏览器本地存储（localStorage）
- 数量限制：与文件夹照片合并后总共最多30张
- 特点：可以随时上传和删除

## 🚀 使用方法

### 上传照片

1. 进入"回忆相册"模块
2. 点击"+ 上传照片"按钮
3. 选择要上传的照片（可以多选）
4. 系统会自动：
   - 检查照片总数是否超过30张
   - 如果超过，会提示并限制数量
   - 自动保存到浏览器本地存储

### 删除照片

1. 将鼠标悬停在上传的照片上
2. 点击右上角的红色删除按钮
3. 确认删除即可

**注意**：只能删除上传的照片，文件夹中的照片需要在代码中配置删除。

## 📊 照片数量计算

- **总照片数** = 文件夹照片数 + 已上传照片数（最多30张）
- **可上传数** = 30 - 文件夹照片数

例如：
- 如果文件夹中有 6 张照片
- 那么最多还可以上传 24 张照片
- 总共最多显示 30 张

## ⚠️ 注意事项

1. **文件大小限制**：每张照片不能超过 10MB
2. **数据存储**：上传的照片保存在浏览器本地，清除浏览器数据会丢失
3. **浏览器兼容**：需要支持 localStorage 的现代浏览器
4. **照片格式**：支持所有常见图片格式（JPG、PNG、GIF等）

## 💡 使用技巧

1. **定期备份**：使用"设置"中的"导出数据"功能备份照片
2. **合理分类**：上传时记得设置照片分类，方便后续筛选
3. **照片描述**：可以编辑照片备注，记录美好瞬间
4. **数量管理**：如果接近30张，可以先删除不需要的照片再上传新的

## 🔧 配置文件夹照片

如果要添加或删除文件夹照片，编辑 `js/gallery.js` 文件：

```javascript
const folderPhotos = [
    { 
        id: 1, 
        url: '3D爱心/images/1.jpg',  // 照片路径
        category: 'daily',             // 分类
        date: '2023.05.20',           // 日期
        location: 'XX餐厅',            // 地点
        note: '第一次一起吃烛光晚餐'    // 备注
    },
    // 添加更多照片...
];
```
